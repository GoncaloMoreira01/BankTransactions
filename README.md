
# BankTransactions - Challenge by Natixis
- Application developed for managing banking transactions with automatic fee calculation.
## Used tecnologies:
 - Java springboot
 - Swagger
 - Maven
 - MySql

### Step-by-step guide to test the application:
 - Get the [repository](https://github.com/GoncaloMoreira01/BankTransactions) of the project and download it
 - Download mySql
 - Configure the "application.properties" according to your database configurations (username and password)
 - Open the project in a IDE;
 - Make sure you have Java installed on your computer
 - Run the BankTransactionsApplication class
 - Create, Update, Delete and Get the transactions List using [Swagger](http://localhost:8080/swagger-ui/index.html)
